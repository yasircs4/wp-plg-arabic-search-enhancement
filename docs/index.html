Search should not punish Arabic spelling variants.

Visitors often type Arabic without diacritics or with different letter forms. This plugin normalizes the query so useful content can still be found.

قران

Finds Quran content

Matches content written as قرآن even when the user omits Hamza.

الاسلام

Handles Alef forms

Connects common variants such as الإسلام and الاسلام.

مكة

Normalizes endings

Improves matching around Taa Marbuta, Yaa, Tatweel, and Tashkeel.

Install in four steps.

The recommended path is the WordPress.org plugin directory so updates are handled by WordPress.

Open plugins

Go to Plugins, then Add New in your WordPress admin.

Search the name

Find Arabic Search Enhancement in the plugin directory.

Install and activate

Click Install Now, then Activate.

Configure search

Open Settings, then Arabic Search, and choose the options your site needs.

Give Arabic visitors fewer dead ends.
