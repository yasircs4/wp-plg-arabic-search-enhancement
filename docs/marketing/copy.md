## One-Line Pitch

Arabic Search Enhancement makes WordPress search more forgiving for Arabic content by matching common spelling, diacritic, Hamza, Alef, Yaa, and Taa Marbuta variations.

## Short Description

Arabic Search Enhancement is a free WordPress plugin that improves Arabic search results by normalizing common Arabic text variations. It helps visitors find content even when they type without diacritics, use a different Alef form, or search with common Hamza and letter variants.

## Long Description

Arabic Search Enhancement helps Arabic WordPress sites reduce failed searches. Standard WordPress search can miss useful content when visitors type Arabic differently from how it appears in posts. This plugin normalizes common Arabic variations during search, including diacritics, Alef forms, Hamza variants, Yaa, Waw, Tatweel, and Taa Marbuta.

The plugin includes admin settings, a self-test section, optional local analytics, Arabic translation files, and right-to-left admin support. Search analytics stay inside the WordPress database and can be disabled.
