## Screenshot Guidelines

- **Size**: 1200x800 pixels (recommended)
- **Format**: PNG or JPG
- **Quality**: High resolution for clarity
- **Content**: Show actual plugin interface, not mockups
- **Language**: Include both English and Arabic examples where applicable

## Usage

These screenshots will be displayed on the landing page to showcase the plugin's functionality and user interface.
