# Arabic Search Enhancement 1.4.8 is live on WordPress.org

Arabic Search Enhancement is now available in the official WordPress Plugin Directory.

Download it here: https://wordpress.org/plugins/arabic-search-enhancement/

## Why this plugin exists

Arabic search can be unforgiving. A visitor may type a word without diacritics, with a different Alef form, or with a common Hamza variation, while the post content uses another valid form. Standard WordPress search can treat those as different text and return no results.

Arabic Search Enhancement helps close that gap by normalizing common Arabic writing variations during search.

For example, a visitor searching for `قران` can still find content written as `قرآن`.

## What it improves

- Arabic diacritics and Tatweel handling
- Alef form normalization
- Hamza variant normalization
- Yaa and Waw variant matching
- Taa Marbuta matching
- Right-to-left admin support
- Local search analytics that stay in your WordPress database
- A self-test screen for checking normalization behavior

## Who should install it

This plugin is useful for Arabic blogs, publishers, schools, mosques, libraries, documentation sites, WooCommerce stores, and any WordPress site where Arabic search needs to feel more natural.

## Privacy

The plugin does not send search data to outside services. Search analytics are optional and stored locally in the WordPress database.
